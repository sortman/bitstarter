The challenge - provide secure desktop computing for corporate users in a world full of unsecure computing devices.

Corporations, educational institutions, non-profits, and government entities rely on their staff and students to use personal and shared computers to accomplish their work. This means using personal laptops, library workstations, home PCs, and a variety of shared computing devices in public spaces. There is little assurance that these devices are properly secured, patched, and managed properly, thus exposing the corporate enterprise to attacks originating - often unknowlingly - from their own employees and members.

The solution - enable trusted computing on untrusted devices using a secure bootable USB drive with built-in authentication using existing hardware credentials (smart card/CAC card).

Bootable USB media that runs only in local memory and loads or reads no software or data on the local device exist in many forms today. The FailSafe bootable USB takes this concept a step further by integrating a smart card reader into the USB hardware device. An enterprise can leverage their existing smart cards used already for identification and access. Only when inserted and activated with the smart card PIN is the user able to fully boot the desktop stored on the attached USB device.
